int DB_find ( const char *  url)

Finds whether or not DB_FILE has url contained in it.

Parameters
urlThe url to search DB_FILE for.
Returns
0 if url is not contained in DB_FILE. 1 if url is contained. -1 if an error occurred.

Example:
