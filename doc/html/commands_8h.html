int Command_fetch ( apr_pool_t *  p,
const char *  url,
int  fetch_only 
)

Downloads a project either through git or curl, and then untars it if needed.

Parameters
pThe apr memory pool to be used.
urlThe url of the project to be fetched.
fetch_onlyWhether or not to only fetch dependencies. 0 to fetch dependencies. 1 to not fetch dependencies.
Returns
0 if nothing additional should be done. 1 if the project needs to be installed. -1 if an error occurred.

Example:

int rc = Command_fetch(p, "https://github.com/foo/bar.git")
if(rc == -1) {
// Handle error...
} else if(rc == 1) {
Command_install(p, "https://github.com/foo/bar.git", NULL, NULL, NULL);
} else {
so_chic();
freak_out();
}
